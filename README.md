A simple and useful multi-purpose bot for your Revolt server!

### Config example:
```js
// config.js
module.exports = {
    token: "YOUR_BOT_TOKEN",
    prefix: "YOUR_BOT_PREFIX",
    dashboardPort: "3000",
    owners: ["YOUR_REVOLT_USER_ID", "another id", "ect.."],
};
```

Just run `npm i` and you are good to go.
